There is a pics folder included but is not needed for it to work, it was just included here for testing purposes.

In order to run the assignment, please click on assignment1.py, and click run. You will then be prompted to select an image. Once you select the image, it will pop up with the GUI where you are able to do the 
histogram operations.

The libraries that were used are:
    PyQT/sys: File exploer for loading image
    MatPlotLib/numpy: Creating histograms and all related operations
    PIL: Loading images
